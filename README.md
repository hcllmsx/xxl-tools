# XXL-Tools

XXL-Tools 是一个用于管理 Ubuntu 系统的工具脚本集合，采用模块化设计，方便扩展和维护。

## 功能列表

1. **密钥登录管理**

   - 开启密钥登录或更新密钥
     > 为当前登录的用户生成 SSH 密钥对。
     > 为已经开启密钥登录的用户更新 SSH 密钥对。
     > 配置 SSH 以禁用密码登录，仅允许密钥登录。
   - 查看并管理公钥
     > 查看系统中所有用户是否开启密钥登录。
     > 删除指定用户的公钥（需要root权限），以禁用密钥登录。
